lhs2texhl by spockz

A helper executable for highlighting code with lhs2tex. Pass an Haskell file to lhs2TeX-hl and it will generate a file with formatting directives to be used with lhs2TeX for you.

For more information, please visit the repository, or look at the release blog post.
